# SOURCE GOVERNANCE RULES V001

## Rule 1 — Direct URL Required

Every public-facing source must include a direct URL when available.

## Rule 2 — Source Class Required

Every source must be classified as government, peer-reviewed, institutional, publisher/textbook, technical, governance, or archive.

## Rule 3 — Boundary Required

Every source must state what it can and cannot support.

## Rule 4 — No Source Substitution

A coding source cannot support a cannabis efficacy claim.

A cannabis clinical source cannot support ICD-10-CM coding methodology.

## Rule 5 — Verification Status Required

Use one of:

- Verified
- Needs Review
- Archived
- Deprecated
- Rejected

## Rule 6 — Claim Traceability

Every medical, coding, or technical claim must trace to a source ID.

## Rule 7 — No Wikipedia

Wikipedia is excluded from the GGTC citation layer.
